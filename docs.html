litestyle css

Litestyle provides basic styles for common HTML elements like headings, paragraphs, lists, links, and images. These styles are designed to be unobtrusive and can be easily overridden with your own custom styles.

buttons

use the .btn class in your button for a default button

use .btn-colorname class for colored buttons eg,

<button class="btn btn-warning">warning</button>

image styles

Litestyle provides you built in image styles the default size for images is 200PX you can use .img to set this size,you can use .img-s,.img-l,img-xl to adjust sizes,.img-round to make your image round,or .img-rectangle to make it rectangular

Both Flexbox And Grid Layout support

litestyle provides both flexbox as well as grid for layout

use .container-flex for activating flexbox

flexbox classes

name use
.container-flex for activating flexbox
.container-flex-align-center sets justify-content and align-items to center
.container-flex-row and . container-flex-column sets flex-direction to row and column

Animations

you can even add animations and hover effects with litestyle

use .animation-3dhover class to add 3d hover effect on the elements you want

.fade-in for fade intro effect

.animation-fade for fading effect on hover
